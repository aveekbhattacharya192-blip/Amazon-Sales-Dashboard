# Amazon Sales Performance Dashboard

## Overview
This project presents an interactive Power BI dashboard built using Amazon sales data. The dashboard provides insights into sales performance, order status distribution, monthly revenue trends, product category performance, and state-wise sales analysis.

## Tools Used
- Power BI
- Microsoft Excel

## Dashboard Insights
- Category-wise Sales Analysis
- Monthly Revenue Trend
- Order Status Distribution
- State-wise Sales Performance
- Total Revenue KPI
- Total Orders KPI

## Files Included
- Amazon Sale Analysis.xlsx (Dataset)
- Amazon_Sales_Dashboard.pbix (Power BI Dashboard)
- Screenshot 2026-06-17 132526.png (Dashboard Preview)

## Key Findings
- Set category generated the highest revenue.
- Maharashtra recorded the highest sales among all states.
- Most orders were successfully shipped.
- Sales peaked during April and remained strong through June.
